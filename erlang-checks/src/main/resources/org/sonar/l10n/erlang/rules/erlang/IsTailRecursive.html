Check if the function is tail recursive or not. Currently it check for direct recursion. A function is tail recursive if the recursive call is a stand alone statement and it is not flowed by other statements:

The following snippets are not tail recursive: 

function([H|T], Acc) ->
   case H of
       1 -> function(T, Acc+1);
       2 -> function(T, Acc-2);
       _ -> 1+function(T, Acc)
   end.
